# Tags

Tags can be used as labels to highlight brief important information.

    <span class="nhs-tag">.nhs-tag</span>
    <span class="nhs-tag nhs-grey">.nhs-tag .nhs-grey</span>
    <span class="nhs-tag nhs-green">.nhs-tag .nhs-green</span>

```html
    <span class="nhs-tag">.nhs-tag</span>
    <span class="nhs-tag nhs-dark-grey">.nhs-tag .nhs-dark-grey</span>
    <span class="nhs-tag nhs-green">.nhs-tag .nhs-green</span>
```
